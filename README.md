# LearnHub - E-commerce Learning Management System

A modern, minimal e-commerce LMS built with pure HTML, CSS, and JavaScript for college project submission.

### 1. **Home Page**
- Hero section with attractive design
- Featured courses showcase
- Trending courses section
- Statistics display
- Call-to-action buttons

### 2. **Courses Page**
- Display all courses with filtering options
- Search functionality
- Category, level, and sorting filters
- Add to wishlist, compare, and buy options

### 3. **Wishlist Feature**
- Add/remove courses from wishlist
- Persistent storage using LocalStorage
- Clear all wishlist items
- Quick actions from wishlist

### 4. **Course Comparison**
- Compare up to 3 courses side-by-side
- Detailed comparison table
- Remove individual courses
- Clear all comparisons
- Direct purchase from comparison

### 5. **Checkout Process**
- Course details review
- Multiple payment methods
- Order summary with calculations
- Form validation
- Success message with redirection

### 6. **Student Dashboard**
- Learning progress tracking
- Course statistics
- Recent activity timeline
- Recommended courses
- Progress update functionality
- Certificate generation (demo)

### 7. **Sell Course (Instructor)**
- Comprehensive course creation form
- Real-time course preview
- Future enhancement notice
- Instructor information section
